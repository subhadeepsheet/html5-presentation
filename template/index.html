The Advanced

Responsive Web

  • What?
  • The approach that suggests that design and development should respond to the user's behavior and environment based on screen size, platform and orientation.
  • Why?
    • Better user experience
    • It's only 1 site!, No need to develop multiple versions for different devices
    • Easy to maintain and develop, it's pure CSS!
  • How?
  • Proportionally size everything
    • Grid/layout
    • Images
    • Font Size

The Catch

Simply: Legacy browsers do not support these features.

Also: This is completely fine.

Graceful degradation

Demo: Rounded corners, drop shadows

Do:

  • Use advanced techniques
  • Degrade gracefully
  • Be okay with degradation
  • This means you, designer!

Don’t:

  • Worry (be happy)
  • Provide overly-complicated fallbacks
  • Over-emphasize this point. Make it status quo. We’ll buy you beers.

The web evolves. We should too.
